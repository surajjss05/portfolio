# 💼 Suraj — Personal Portfolio Website

A modern, responsive, and visually stunning **personal portfolio website** built to showcase skills, projects, and accomplishments as a web developer. Designed with a sleek dark theme, gradient accents, and interactive elements to capture visitors' attention.

🔴 **Live Demo:** [https://s3portfoli.netlify.app/](https://s3portfoli.netlify.app/)

## 🚀 Features

- 🏠 **Home Page** — Captivating headline with animated typing effect and developer intro
- 👤 **About Me** — Detailed background, education, and professional experience
- 🛠️ **Skills Section** — Visual showcase of technical skills and proficiencies
- 🗂️ **Portfolio / Projects** — Highlights 25+ projects with descriptions and links
- 📄 **Resume** — Downloadable resume section
- 📬 **Contact** — Contact form and social media links
- 🌗 **Dark / Light Mode Toggle** — Switch between themes seamlessly
- 📊 **Live GitHub Stats** — Real-time commit and contribution stats
- 📱 **Fully Responsive** — Optimized for all screen sizes and devices
- ✨ **Smooth Animations** — Elegant transitions and hover effects throughout

## ⚙️ How to Use

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/portfolio.git
   ```
2. **Navigate to the project folder**
   ```bash
   cd portfolio
   ```
3. **Open `index.html` in your browser**
   ```bash
   open index.html
   ```

## 🌐 Deployment

This portfolio is deployed using **Netlify** for fast and reliable hosting.

[![Netlify Status](https://api.netlify.com/api/v1/badges/your-badge-id/deploy-status)](https://s3portfoli.netlify.app/)

To deploy your own version:
1. Push your code to GitHub
2. Connect your repo to [Netlify](https://netlify.com)
3. Click **Deploy Site** — done! 🚀
